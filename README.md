# VibeClub

VibeClub is a music-first social platform focused on performance, movement and creative expression.

## Core Concept
- Music is central to every experience
- Performance is not limited to dance
- Movement can be body, voice, instrument or sound
- Community interaction happens through Moves, Challenges and Battles

## Main Modules
- Feed Global (aggregator)
- VibeMoves (Moves, Challenges, Battles)
- VibeSessions
- NewVibes
- VibeCasts
- Shows
